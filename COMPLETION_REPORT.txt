================================================================================
DEPLOYMENT
================================================================================

Docker Compose (recommended):
  docker compose up -d
  # App available at http://localhost:8080
  # Migrations run automatically on startup

The Dockerfile builds the Vue frontend and serves it via FastAPI's static file
handler. The backend (uvicorn) runs directly without Gunicorn. PostgreSQL 16
with PostGIS runs as a separate container.
