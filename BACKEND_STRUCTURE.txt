KEY FEATURES IMPLEMENTED:
=========================

MODELS:
-------
✓ Destination: UUID PK, full location data, custom fields (JSONB), cover image
✓ Link: Referenced URLs with metadata
✓ Media: File management with FK to destinations/journal_entries
✓ JournalEntry: Travel journal with dates and ratings
✓ CustomFieldDefinition: User-defined extensible fields
✓ AppSetting: Key-value settings for AI config and prompt template

INDEXES & PERFORMANCE:
---------------------
✓ GIN index on tags array
✓ Full-text search tsvector index (GIN)
✓ GIST index on PostGIS location geometry
✓ Standard B-tree indexes on status, country, region, city, priority

  - Set cover image

Media (5):
  - Upload multiple files with thumbnail generation

RUNNING THE APPLICATION:
========================

Docker (recommended):
  docker compose up -d
  # App available at http://localhost:8080
  # Migrations run automatically on startup

Development:
  # Backend
  cd backend
  python -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  alembic upgrade head
  uvicorn app.main:app --reload

  # Frontend
  cd frontend
  npm install
  npm run dev
